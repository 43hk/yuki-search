# AGENTS.md

## 本地工作约束

- 在本地工作时不要进行任何运行和验证操作，例如不要运行写好的代码、不要启动服务、不要执行构建或测试。
- 只修改文件，运行和验证由项目维护者手动完成。

## 项目理解

- 这是一个静态搜索首页，入口在 `src/index.html`，样式在 `src/style.css`，交互逻辑在 `src/app.js`。
- 搜索引擎配置位于 `ENGINES`，当前只包含 Google 和 Bing。
- 搜索候选优先请求 `/suggest` 代理；代理不可用时，前端会尝试直连/JSONP 兜底。
- 壁纸来自 `https://uapis.cn/api/v1/random/image`，桌面端使用 `type=pc`，移动端使用 `type=mb`。
- 壁纸按日期和设备类型存入 `localStorage`，并尽量写入浏览器 Cache API；同一天、同设备类型优先复用缓存。

## 未来建议

- 如果候选接口要稳定上线，建议在 Nginx 中保留 `/suggest` 反向代理，避免浏览器直连接口被 CORS 或第三方策略影响。
- 如果需要更稳定的首屏速度，建议增加 Service Worker，预缓存当天壁纸并控制旧壁纸清理策略。
- 如果后续增加更多搜索引擎，建议把候选接口也做成与 `ENGINES` 同级的配置，避免逻辑分散。
- 移动端适配优先检查搜索框、候选栏、底部按钮和横屏短高度场景。
